# ✨ 나랑 닮은 디즈니 캐릭터 찾기 (Disney Character Finder)

8가지 간단한 질문을 통해 당신의 내면과 가장 닮은 디즈니 캐릭터를 찾아주는 웹 애플리케이션입니다.

## 🚀 주요 기능
- **성격 유형 퀴즈**: 8가지 질문으로 사용자의 성향(외향/내향, 리더십, 창의성 등)을 분석합니다.
- **캐릭터 매칭**: 엘사, 모아나, 아리엘, 미키 마우스 등 12종의 다양한 디즈니 캐릭터 중 가장 잘 어울리는 캐릭터를 매칭합니다.
- **상세 결과 제공**: 캐릭터의 특성, 설명, 그리고 성격 키워드(#태그)를 함께 보여줍니다.
- **안정적인 이미지**: 위키미디어(Wikimedia) 기반의 신뢰도 높은 이미지 소스를 사용합니다.
- **반응형 디자인**: 모바일과 데스크톱 모두에서 쾌적하게 이용할 수 있는 심플하고 세련된 UI를 제공합니다.

## 🛠 사용된 기술
- **HTML5**: 구조설계 및 시맨틱 태그 활용
- **CSS3**: 현대적인 카드 레이아웃 및 둥근 프레임 스타일링
- **JavaScript (Vanilla JS)**: 퀴즈 로직 처리 및 동적 콘텐츠 렌더링
- **Wikimedia Images**: 캐릭터 시각화를 위한 이미지 소스

## 📂 프로젝트 구조
- `index.html`: 애플리케이션의 모든 로직과 스타일링이 포함된 메인 파일
